Important Information

When converting a user to a student, the following changes will occur:

  • User role will be updated to "student"
  • Student profile record will be created
  • User will gain access to student portal
  • All existing data will be preserved
  • User will be notified via email
Note: This action cannot be undone automatically. If you need to revert a user's status, please contact the system administrator.
